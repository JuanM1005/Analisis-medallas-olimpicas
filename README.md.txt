# 🏅 Análisis de Medallas Olímpicas

Este proyecto realiza un Análisis exploratorio de datos (EDA) sobre un conjunto de datos de medallas olímpicas, utilizando Python y Jupyter Notebook.  
El objetivo es identificar patrones, tendencias históricas y los países con mejor desempeño en diferentes ediciones de los Juegos Olímpicos.

---

## 📂 Contenido del repositorio
- `Análisis exploratorio de datos_Aguirre Mares.ipynb`: Notebook principal con el análisis.
- `data/`: Carpeta con los datos utilizados.
- `requirements.txt`: Lista de librerías necesarias para ejecutar el proyecto.

---

## 🚀 Tecnologías utilizadas
- Python 3.13
- Pandas
- Jupyter Notebook
